더욱 편리한 추가 기능

두 가지 추가 기능으로 ERP 사용을 더욱 편리하고 안전하게 만들어 보세요.

⚡️

구글 로그인 활성화

로그인 화면의 [구글 로그인 활성화] 옵션을 체크하세요. 다음부터는 ERP를 실행하면 로그인 화면 없이 바로 메인 화면으로 접속되어 더욱 빠르게 업무를 시작할 수 있습니다.

🗑️

구글 정보 삭제

ERP 메인 화면 우측 상단의 사용자 메뉴에서 [구글정보삭제]를 클릭하여 현재 컴퓨터의 로그인 정보를 즉시 삭제할 수 있습니다. 공용 PC 사용 시 보안을 위해 꼭 사용해 주세요.

🚨 중요 보안 안내

개인 PC가 아닌 여러 사람이 함께 사용하는 **공용 컴퓨터**에서 Google 계정으로 로그인한 경우, 업무 종료 시 반드시 ERP 메인 화면의 **[구글정보삭제]** 버튼을 클릭하여 개인정보를 안전하게 보호해 주세요.
